## üÜò **Troubleshooting**

### **Common Issues**

**"Please select a class" Error**
- **Cause**: No class selected in dropdown
- **Solution**: Choose a class from the dropdown menu

**"No data found" Message**
- **Cause**: No attendance data for selected criteria
- **Solution**: Adjust date range or remove some filters

**Report not generating**
- **Cause**: Network issues or server problems
- **Solution**: Check internet connection and try again

**CSV download not working**
- **Cause**: Browser popup blocker or download restrictions
- **Solution**: Allow popups and check download permissions

**Charts not displaying**
- **Cause**: JavaScript disabled or browser compatibility
- **Solution**: Enable JavaScript or use updated browser

### **Performance Tips**
1. **Limit Date Range**: Use shorter periods for faster generation
2. **Specific Filters**: Apply filters to reduce data volume
3. **Optimal Timing**: Generate reports during off-peak hours
4. **Browser Cache**: Clear cache if experiencing issues
5. **Stable Connection**: Ensure stable internet for large reports
